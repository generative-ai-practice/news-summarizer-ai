# Increased file size limits and expanded inputs support in Gemini API

## Key Points
- Gemini APIは、Google Cloud Storage (GCS) オブジェクト登録と、HTTPS/署名付きURLからのファイル入力をサポートするようになった。
- これにより、開発者は既存のストレージからデータを再アップロードすることなく、GCSファイルや公開/署名付きURL (AWS S3, Azure Blob Storageなど) を通じて直接データを取り込める。
- インラインデータの最大ファイルサイズ制限が20MBから100MBに引き上げられ、より大きな画像や短い音声クリップに対応できるようになった。
- これらの機能強化は、データ取り込みの柔軟性を高め、AIアプリケーションの迅速な構築と本番環境へのスケールアップを可能にする。
