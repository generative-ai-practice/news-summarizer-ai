# Supporting the web with new features and partnerships


## Summary
Google検索は、ユーザーが好みの情報源とつながり、ウェブを探索するのに役立つ新機能とパートナーシップを発表しています。ユーザーは「トップニュース」をカスタマイズして、お気に入りの情報源からのコンテンツをより多く見ることができるようになりました。Geminiアプリでは、ニュース購読からのリンクがハイライト表示され、AIモードではインラインリンクが文脈に応じた紹介文と共に増加し、ウェブ探索を支援します。

さらに、GoogleはAIがよりエンゲージメントの高い読者を獲得するのにどのように役立つかを探るため、世界中のニュース出版社との商業パートナーシッププログラムを試験運用しています。このプログラムでは、AIを活用した記事の概要やオーディオブリーフィングなどがGoogleニュースでテストされ、コンテンツへのより深いエンゲージメントを促進します。

## Key Points
- Googleは、ユーザーとウェブサイト向けの新機能とパートナーシップを通じてウェブを支援しています。
- 「Preferred Sources」機能により、ユーザーはお気に入りのサイトから「トップニュース」でより多くのコンテンツをカスタマイズできるようになります。
- GeminiアプリのAI概要では、ニュース購読からのリンクがハイライト表示されるようになります。
- AIモードのインラインリンクが増加し、その価値を説明する文脈に応じた紹介が追加されます。
- Googleは、AIが読者エンゲージメントを促進する方法を模索するため、ニュース出版社とのAIパートナーシッププログラムを試験運用しています。
