# Donating the Model Context Protocol and establishing the Agentic AI Foundation

**Published:** 2025-12-09
**URL:** https://www.anthropic.com/news/donating-the-model-context-protocol-and-establishing-of-the-agentic-ai-foundation
**Source:** news
**Language:** ja

## Summary
Anthropicは本日、「Model Context Protocol（MCP）」をLinux Foundation傘下に新設された「Agentic AI Foundation（AAIF）」に寄贈したことを発表しました。AAIFはAnthropic、Block、OpenAIが共同設立し、Google、Microsoft、Amazon Web Services（AWS）、Cloudflare、Bloombergなどの大手企業が支援する目的基金であり、Agentic AIの透明性、協調性、公共の利益に資する発展を目指しています。

MCPは1年前に、AIアプリケーションを外部システムに接続するための普遍的なオープン標準として導入されました。以来、ChatGPT、Gemini、Microsoft Copilotなどの主要なAI製品に採用され、10,000以上の活動中の公開MCPサーバーが存在し、PythonやTypeScript向けの公式SDKは月間9,700万以上のダウンロードを記録するなど、目覚ましい普及を遂げています。Anthropicは引き続きMCPの成長に投資し、ClaudeのコネクタやAPIにおける高度なツール使用機能を提供しています。

今回のMCPのLinux Foundationへの寄贈は、オープンソース、コミュニティ主導、ベンダーニュートラルな標準であり続けるというAnthropicのコミットメントをさらに強化するものです。AAIFの下で、MCPはBlockのgooseやOpenAIのAGENTS.mdといった創設プロジェクトに加わり、Agentic AIエコシステム全体のイノベーションを促進し、これらの基盤技術が中立的かつオープンでコミュニティ主導であることの保証を目指します。

## Key Points
- Anthropicは、AIアプリケーションと外部システムを接続するオープン標準である「Model Context Protocol (MCP)」を、新設の「Agentic AI Foundation (AAIF)」に寄贈した。
- AAIFはLinux Foundationの傘下にある目的基金で、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、AWSなどが支援している。
- MCPは導入から1年でChatGPT、Gemini、Microsoft Copilotなど多くのAI製品に採用され、10,000以上の公開サーバーと月間9,700万以上のSDKダウンロードを達成している。
- Anthropicは引き続きMCPの成長に投資しており、Claudeの75以上のコネクタやAPIでのTool SearchおよびProgrammatic Tool Calling機能を提供している。
- この寄贈は、MCPが中立でオープン、コミュニティ主導の標準としてAgentic AIエコシステムのイノベーションを促進するというAnthropicのコミットメントを示すものである。
