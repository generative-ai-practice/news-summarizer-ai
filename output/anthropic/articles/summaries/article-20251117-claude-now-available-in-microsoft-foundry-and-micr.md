# Claude now available in Microsoft Foundry and Microsoft 365 Copilot

## Key Points
- Claude Sonnet 4.5、Haiku 4.5、Opus 4.1モデルがMicrosoft Foundryでパブリックプレビューとして利用可能となり、Azure顧客は生産アプリケーションやエンタープライズエージェントを構築できます。
- Microsoft 365 CopilotのResearcherエージェントに加え、ExcelのAgent ModeでもClaudeがプレビューで利用可能になり、スプレッドシートの生成やデータ分析などがExcel内で直接行えます。
- 既存のMicrosoft FoundryやMicrosoft 365 Copilotに投資している企業は、個別のベンダー契約や請求システムなしでClaudeのAI機能を採用できます。
- 開発者はFoundryのAPI、ツール、ワークフローを通じてClaudeを迅速に導入でき、Python、TypeScript、C# SDK、Microsoft Entra認証に対応しています。
- Sonnet 4.5はコーディングと複雑なエージェント構築、Haiku 4.5は速度とコスト効率が求められる高ボリュームアプリケーション、Opus 4.1は専門的な推論タスクにそれぞれ最適化されています。
