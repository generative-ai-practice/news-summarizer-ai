# Claude for Nonprofits


## Summary
Anthropicは、国際的な慈善活動ムーブメントであるGivingTuesdayと提携し、「Claude for Nonprofits」プログラムを開始しました。このプログラムは、限られたリソースの中で社会の最も困難な問題に取り組む非営利団体が、その影響力を最大化できるよう支援することを目的としています。てんかん財団（Epilepsy Foundation）や国際救助委員会（International Rescue Committee）、IDinsightなどの複数の非営利団体が既にClaudeを活用し、24時間体制のサポート提供、人道支援におけるデータ分析の高速化、作業効率の16倍向上といった成果を上げています。

この新しいプログラムには、非営利団体向けに最大75%の割引でClaudeのTeamおよびEnterpriseプランへのアクセスが含まれます。これにより、助成金申請やプログラム分析に適したClaude Sonnet 4.5や、高速処理が可能なClaude Haiku 4.5を利用できます。また、Blackbaud、Candid、Benevityといった非営利団体に特化したツールとClaudeを連携させるための新しいオープンソースコネクターも導入され、既存のビジネスツールとの統合も可能です。

さらに、AnthropicはGivingTuesdayとの協力のもと、AI初心者でも効果的にAIを利用できるよう設計された無料のオンラインコース「AI Fluency for Nonprofits」を提供します。このカリキュラムは、助成金申請書の作成、プログラム評価、寄付者エンゲージメント、組織効率化など、非営利団体の主要な業務におけるAIの活用方法に焦点を当てています。Constellation Fund、Robin Hood、Tipping Point Communityなどのパートナーとのパイロットプログラムを通じて、助成先団体が助成金提案の作成、プログラム影響分析、大規模な寄付者対応、ボード資料やコンプライアンス文書の生成などにおいて、Claudeをより効果的に活用できるよう支援しています。

## Key Points
- AnthropicはGivingTuesdayと提携し、非営利団体がAIを効率的に活用し、社会的影響を最大化できるよう「Claude for Nonprofits」プログラムを開始。
- 非営利団体は、ClaudeのTeamおよびEnterpriseプランを最大75%割引で利用でき、Claude Sonnet 4.5、Haiku 4.5、Opus 4.5（Enterprise版のみ）へのアクセスが可能となる。
- Blackbaud、Candid、Benevityといった非営利団体向け主要ツールとの新しいオープンソースコネクターを導入し、既存のビジネスツール（Microsoft 365、Google Workspaceなど）との連携も強化。
- 非営利団体向けにAIの効率的な利用法を学ぶ無料のオンラインコース「AI Fluency for Nonprofits」と、具体的なユースケースのガイドを提供。
- てんかん財団、国際救助委員会、IDinsightなどの非営利団体が既にClaudeを導入し、24時間サポート、迅速なデータ分析、作業効率の向上といった具体的な成果を報告している。
