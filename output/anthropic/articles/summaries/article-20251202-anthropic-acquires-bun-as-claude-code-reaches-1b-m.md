# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、Claude Codeのさらなる加速のため、画期的なJavaScriptランタイムであるBunを買収しました。
- AnthropicのAIモデルであるClaude Codeは、2025年5月の提供開始からわずか6ヶ月後の2025年11月に、年間収益10億ドルという重要なマイルストーンを達成しました。
- Bunは、JavaScriptおよびTypeScript開発者向けの高速なオールインワンツールキット（ランタイム、パッケージマネージャー、バンドラー、テストランナー）であり、最新のソフトウェアエンジニアリングにおける速度とパフォーマンスを再定義しています。
- この買収により、Claude Codeのユーザーはパフォーマンスの高速化、安定性の向上、および新機能の恩恵を受けられるようになります。
- Bunは今後もオープンソース（MITライセンス）として維持され、AnthropicはJavaScriptおよびTypeScript開発者にとって最高のランタイムであり続けるよう投資を続けます。
