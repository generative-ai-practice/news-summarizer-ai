# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- AnthropicはJavaScriptランタイム「Bun」を買収しました。これはClaude Codeのさらなる加速と、開発者向けのAI主導ソフトウェアエンジニアリングインフラの強化を目的としています。
- AnthropicのAIモデルであるClaude Codeは、公開からわずか6ヶ月で年間収益10億ドルを達成し、その優れた能力と市場での成功を示しました。
- Bunは、極めて高速なJavaScriptランタイムであり、パッケージマネージャー、バンドラー、テストランナーを統合したオールインワンのツールキットとして、現代のソフトウェア開発のスピードと性能を再定義しています。
- Bunの買収により、Claude Codeのユーザーはパフォーマンス、安定性、新機能の向上を享受でき、AnthropicはBunをオープンソースとして維持し、その開発に引き続き投資します。
- この戦略的買収は、Anthropicが技術的卓越性を高め、エンタープライズAI分野でのリーダーシップを強化し、その原則とミッションに沿った機会を追求する姿勢を反映しています。
