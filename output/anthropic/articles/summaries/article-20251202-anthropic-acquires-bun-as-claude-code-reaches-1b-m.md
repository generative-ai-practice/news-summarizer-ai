# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、JavaScriptランタイム「Bun」を買収しました。
- Claude Codeは、提供開始からわずか6ヶ月で年間売上10億ドルを達成しました。
- Bunは、ランタイム、パッケージマネージャー、バンドラー、テストランナーを兼ね備えたオールインワンツールキットであり、JavaScriptおよびTypeScript開発の速度とパフォーマンスを向上させます。
- この買収により、Claude Codeのパフォーマンス向上、安定性強化、および新機能の提供が期待されます。
- Bunは今後もオープンソース（MITライセンス）として維持され、JavaScriptおよびTypeScript開発者向けに継続的な投資が行われます。
