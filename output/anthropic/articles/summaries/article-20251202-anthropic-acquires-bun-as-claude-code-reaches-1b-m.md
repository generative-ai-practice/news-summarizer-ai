# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Claude Codeは、2025年5月の一般提供開始からわずか6ヶ月で年間収益10億ドルを達成する重要なマイルストーンに到達しました。
- Anthropicは、Claude Codeをさらに加速させるため、画期的なJavaScriptランタイムであるBunを買収しました。
- Bunは、2021年にJarred Sumnerによって設立された、JavaScript/TypeScript開発向けの高速なオールインワンツールキットです。
- この買収により、Claude Codeのパフォーマンス向上、安定性強化、および新機能が期待され、Bunは引き続きオープンソース（MITライセンス）として投資されます。
- Bunは月間700万回以上のダウンロードとGitHubで82,000以上のスターを獲得しており、AI主導のソフトウェアエンジニアリングに不可欠なインフラとなっています。
