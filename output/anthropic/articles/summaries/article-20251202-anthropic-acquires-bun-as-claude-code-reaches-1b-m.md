# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Claude Codeは、公開からわずか6ヶ月で年間収益10億ドルという重要なマイルストーンを達成しました。
- Anthropicは、Claude Codeのさらなる加速を目指し、画期的なJavaScriptランタイムであるBunを買収しました。
- Bunは、ランタイム、パッケージマネージャー、バンドラー、テストランナーを統合したオールインワンのツールキットであり、JavaScript開発の速度とパフォーマンスを再定義しています。
- この買収により、Claude Codeはより高速なパフォーマンス、安定性の向上、および新機能を得ることが期待され、開発者体験が改善されます。
- Bunは引き続きオープンソースでMITライセンスを維持し、AnthropicはJavaScriptおよびTypeScript開発者向けの主要なツールとしての投資を継続します。
