# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- AnthropicはJavaScriptランタイム「Bun」を買収しました。これにより、Claude Codeの機能強化とAI主導のソフトウェア開発加速を目指します。Bunは速度とパフォーマンスを再定義するオールインワンツールキットです。
- Claude Codeは2025年5月の一般公開からわずか6ヶ月で、年間経常収益10億ドルという顕著なマイルストーンを達成しました。
- Bunの買収は、Anthropicの技術的卓越性を高め、エンタープライズAI分野でのリーダーシップを強化する戦略的な動きです。
- Bunは今後もオープンソース（MITライセンス）として維持され、JavaScriptおよびTypeScript開発者向けのランタイム、バンドラー、パッケージマネージャー、テストランナーとして引き続き投資されます。
