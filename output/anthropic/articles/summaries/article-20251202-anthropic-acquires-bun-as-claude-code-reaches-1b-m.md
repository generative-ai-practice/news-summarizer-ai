# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、AIモデルClaude Codeが公開からわずか6ヶ月で年間収益10億ドル（run-rate revenue）を達成したことを発表した。
- Claude Codeのさらなる加速のため、Anthropicは画期的なJavaScriptランタイムであるBunを買収した。
- Bunは2021年に設立されたオールインワンのJavaScriptツールキットで、JavaScriptおよびTypeScript開発の速度とパフォーマンスを大幅に向上させる。
- この買収により、Claude Codeのパフォーマンス向上、安定性強化、および新機能の追加が期待される。
- Bunは引き続きオープンソース（MITライセンス）として維持され、月間700万回以上のダウンロードとGitHubで82,000以上のスターを獲得している。
