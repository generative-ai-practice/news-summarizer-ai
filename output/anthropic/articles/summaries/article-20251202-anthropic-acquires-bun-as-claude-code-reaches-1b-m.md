# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、開発者、スタートアップ、企業向けのAIモデルであるClaude Codeが、一般公開からわずか6ヶ月で年間収益10億ドルに達したと発表しました。
- この発表と同時に、Anthropicは画期的なJavaScriptランタイムであるBunを買収したことを明らかにしました。
- Bunは2021年にJarred Sumnerによって設立され、ランタイム、パッケージマネージャー、バンドラー、テストランナーを統合したオールインワンツールキットとして、JavaScriptおよびTypeScript開発の速度とパフォーマンスを劇的に向上させます。
- この買収により、Claude Codeの性能、安定性、および新機能が向上し、Anthropicは引き続きBunをオープンソースおよびMITライセンスで提供し、投資を継続します。
- Bunは月間700万以上のダウンロードとGitHubで82,000以上のスターを獲得しており、MidjourneyやLovableといった企業で採用され、AI主導のソフトウェアエンジニアリングに不可欠なツールとなっています。
