# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、JavaScriptランタイム「Bun」を買収し、Claude Codeのさらなる加速を目指します。
- Claude Codeは、2025年5月の一般提供開始からわずか6ヶ月で年間収益10億ドルを達成しました。
- Bunは、JavaScriptおよびTypeScript開発の速度とパフォーマンスを向上させるオールインワンツールキットであり、AI主導のソフトウェアエンジニアリングの重要なインフラです。
- この買収は、Anthropicが技術的卓越性を強化し、エンタープライズAIのリーダーとしての地位を固めるための戦略的な動きです。
- Bunは引き続きオープンソース（MITライセンス）として提供され、JavaScriptおよびTypeScript開発者向けのランタイム、バンドラー、パッケージマネージャー、テストランナーとして投資が継続されます。
