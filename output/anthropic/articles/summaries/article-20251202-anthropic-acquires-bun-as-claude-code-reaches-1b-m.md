# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- AnthropicのClaude Codeは、一般公開からわずか6ヶ月で年間収益10億ドルのマイルストーンを達成しました。
- この成功を受けて、AnthropicはJavaScriptランタイムであるBunを買収し、Claude Codeのさらなる加速と開発者向けAIインフラの強化を図ります。
- Bunは、その高速性、オールインワンのツールキット（ランタイム、パッケージマネージャー、バンドラー、テストランナー）により、AI主導のソフトウェア開発に不可欠なインフラとなっています。
- Bunの買収により、Claude Codeのユーザーはより高速なパフォーマンス、安定性の向上、そして新機能を利用できるようになります。
- Bunは今後もオープンソースおよびMITライセンスで提供され、JavaScriptおよびTypeScript開発者向けの主要なツールとしてAnthropicが継続して投資を行います。
