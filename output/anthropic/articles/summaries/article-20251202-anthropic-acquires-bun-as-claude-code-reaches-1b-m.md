# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、Claude Codeが公開からわずか6ヶ月で年換算収益10億ドルを達成したことを発表し、その加速のためJavaScriptランタイム「Bun」を買収しました。
- Bunは2021年にJarred Sumnerによって設立され、JavaScriptおよびTypeScript開発の速度とパフォーマンスを劇的に向上させるオールインワンツールキットです。
- この買収により、Claude Codeのユーザーは、より高速なパフォーマンス、安定性の向上、そして新機能の恩恵を受けることになります。
- Anthropicの最高製品責任者であるMike Krieger氏は、Bunの買収がAnthropicの技術的卓越性を強化し、AI採用の指数関数的成長に対応するためのインフラ構築を可能にすると述べました。
- Bunは引き続きオープンソース（MITライセンス）として提供され、AnthropicはJavaScriptおよびTypeScript開発者にとって最適なランタイム、バンドラー、パッケージマネージャー、テストランナーとしてのBunへの投資を継続します。
