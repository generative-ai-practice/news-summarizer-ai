# Anthropic acquires Bun as Claude Code reaches $1B milestone

## Key Points
- Anthropicは、Claude Codeのさらなる加速のために画期的なJavaScriptランタイム「Bun」を買収しました。
- Claude Codeは2025年5月の一般提供開始からわずか6ヶ月で年間収益10億ドルを達成しました。
- Bunは、ランタイム、パッケージマネージャー、バンドラー、テストランナーを兼ね備えたオールインワンのツールキットであり、AI主導のソフトウェアエンジニアリングにとって不可欠なインフラです。
- この買収により、Claude Codeのユーザーはより高速なパフォーマンス、安定性の向上、および新機能を利用できるようになります。
- Bunは引き続きオープンソースでMITライセンスを維持し、AnthropicはJavaScriptおよびTypeScript開発者向けのツールとしての投資を継続します。
