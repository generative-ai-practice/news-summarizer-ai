# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、AIアプリケーションを外部システムに接続するためのオープン標準であるModel Context Protocol (MCP)を、Linux Foundation傘下の新組織Agentic AI Foundation (AAIF)に寄贈しました。
- AAIFはAnthropic、Block、OpenAIが共同設立し、Google、Microsoft、AWS、Cloudflare、Bloombergの支援を受け、エージェントAIの透明性、協力、公益に基づいた進化を目的としています。
- MCPは導入から1年で10,000を超える公開サーバーに採用され、ChatGPT、Gemini、Microsoft Copilotなどの主要AI製品や、AWS、Google Cloudなどのインフラプロバイダーによってサポートされています。
- MCPはClaudeのコネクタや高度なツール利用機能、非同期操作を含む2025年11月25日の仕様リリースで機能強化され、PythonおよびTypeScript向けSDKは月間9,700万回以上ダウンロードされています。
- MCPは、BlockのgooseやOpenAIのAGENTS.mdと共にAAIFの設立プロジェクトとなり、エージェントAIエコシステムのイノベーションを促進し、基盤技術の中立性、オープン性、コミュニティ主導性を確保します。
