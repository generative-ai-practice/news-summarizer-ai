# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、AIアプリケーションと外部システムを接続するオープン標準であるModel Context Protocol (MCP) を、新設されたAgentic AI Foundation (AAIF) に寄贈しました。
- AAIFはLinux Foundationの傘下にあるファンドで、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、AWS、Cloudflare、Bloombergなどの主要企業が支援しています。
- MCPは、導入後1年で10,000以上の公開サーバーで利用され、ChatGPTやMicrosoft Copilotなどの主要AI製品に採用されるなど、広く普及しています。
- この寄贈により、MCPはオープンソース、コミュニティ主導、ベンダー中立の原則を維持し、そのガバナンスモデルは変更されません。
- 本寄贈は、Agentic AIエコシステム全体でのイノベーションを促進し、基盤技術の中立性とオープン性を保証するというAnthropicのコミットメントを示します。
