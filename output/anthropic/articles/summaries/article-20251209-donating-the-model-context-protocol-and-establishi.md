# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、AIアプリケーションを外部システムに接続するためのオープン標準であるModel Context Protocol (MCP) を、Linux Foundation傘下のAgentic AI Foundation (AAIF) に寄贈しました。
- Agentic AI Foundation (AAIF) は、Anthropic、Block、OpenAIが共同で設立し、Google、Microsoft、Amazon Web Services (AWS)、Cloudflare、Bloombergが支援しています。
- MCPは過去1年間で広範に採用され、10,000を超えるアクティブなパブリックサーバーがあり、ChatGPTやMicrosoft Copilotなどの主要なAI製品に導入されています。
- 今回の寄贈は、MCPがオープンソース、コミュニティ主導、ベンダー中立性を維持するというAnthropicのコミットメントを強化するものです。
- AAIFは、エージェントAIエコシステム全体のイノベーションを促進し、基盤となるテクノロジーが中立的、オープン、コミュニティ主導であり続けることを目指します。
