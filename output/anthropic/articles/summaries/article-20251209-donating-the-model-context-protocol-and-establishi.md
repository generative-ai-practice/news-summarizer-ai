# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、AIアプリケーションを外部システムに接続するためのユニバーサルなオープン標準であるModel Context Protocol (MCP) を、Agentic AI Foundation (AAIF) に寄贈しました。
- AAIFは、Linux Foundation傘下のファンドで、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、Amazon Web Services (AWS) などが支援しています。
- MCPは発表から1年で10,000以上の公開サーバーに採用され、ChatGPT、Cursor、Geminiなどの人気AI製品に導入されるなど、広範な普及を達成しました。
- MCPは、オープンソース、コミュニティ主導、ベンダーニュートラルな姿勢を維持し、そのガバナンスモデルは今後もコミュニティの意見と透明な意思決定を優先します。
- AAIFへのMCP寄贈により、エージェントAIエコシステム全体でのイノベーションを促進し、これらの基盤技術が中立的でオープン、かつコミュニティ主導であり続けることを目指します。
