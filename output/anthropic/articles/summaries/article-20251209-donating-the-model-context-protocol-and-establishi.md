# Donating the Model Context Protocol and establishing the Agentic AI Foundation


## Summary
Anthropicは、AIアプリケーションと外部システムを接続するためのユニバーサルなオープン標準であるModel Context Protocol (MCP)を、Linux Foundation傘下の新しいAgentic AI Foundation (AAIF)に寄付しました。AAIFは、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、Amazon Web Services (AWS)、Cloudflare、Bloombergなどの主要企業が支援する組織です。AAIFは、エージェントAIが透明性、協調性、公共の利益に資する形で発展することを促進することを目指しています。

MCPは、発表から1年で10,000を超える公開サーバーで利用され、ChatGPT、Gemini、Microsoft Copilotなどの主要AI製品や、AWS、Google Cloud、Microsoft Azureといった企業向けインフラストラクチャに採用されるなど、目覚ましい普及を遂げています。AnthropicはClaudeの75以上のコネクタ（MCP利用）やAPIのツール検索機能、プログラムツール呼び出し機能を通じてMCPの発展を支援し続けています。また、MCPはコミュニティ主導の公式レジストリを持ち、非同期操作やサーバーIDなどの新機能を備えた最新の仕様がリリースされています。主要なプログラミング言語向けの公式SDKは、PythonとTypeScriptで月間9700万回以上ダウンロードされています。

今回の寄付は、MCPがオープンソース、コミュニティ主導、ベンダーニュートラルな標準であり続けるというAnthropicのコミットメントを強化するものです。MCPはBlockのgooseやOpenAIのAGENTS.mdと共にAAIFの創設プロジェクトとなり、エージェントAIエコシステム全体のイノベーションを促進し、これらの基盤技術が中立的かつオープンであり続けることを保証します。

## Key Points
- Anthropicは、AIアプリケーションの外部システム接続標準であるModel Context Protocol (MCP)をAgentic AI Foundation (AAIF)に寄付した。
- Agentic AI Foundation (AAIF)は、Linux Foundation傘下の組織で、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、AWSなどが支援している。
- MCPは、発表から1年で10,000以上のサーバーで採用され、ChatGPT、Gemini、Microsoft Copilotなどの主要AI製品やクラウドインフラに広く普及している。
- AnthropicはClaudeのコネクタやAPI機能を通じてMCPの成長に貢献しており、MCPは非同期操作やサーバーIDなどの新機能を導入し、主要言語のSDKが月間9700万回以上ダウンロードされている。
- この寄付により、MCPがオープンソース、コミュニティ主導、ベンダーニュートラルな標準として、エージェントAIエコシステムのイノベーションを促進するというAnthropicのコミットメントが強化される。
