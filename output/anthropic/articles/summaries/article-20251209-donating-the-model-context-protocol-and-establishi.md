# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、AIアプリケーションを外部システムに接続するオープン標準であるModel Context Protocol（MCP）を、Linux Foundation傘下のAgentic AI Foundation（AAIF）に寄贈しました。
- AAIFは、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、AWS、Cloudflare、Bloombergが支援する団体で、エージェントAIの透明性とオープン標準の発展を目指します。
- MCPは過去1年間で広く採用され、10,000以上の公開サーバーで利用されており、ChatGPT、Cursor、Gemini、Microsoft Copilotなどの主要なAI製品に統合されています。
- MCPはオープンソース、コミュニティ主導、ベンダー中立性を維持することにコミットしており、今回の寄贈はこのコミットメントをさらに強化するものです。
- MCPは、Blockの「goose」やOpenAIの「AGENTS.md」と共にAAIFの設立プロジェクトとなり、エージェントAIエコシステム全体のイノベーションを促進します。
