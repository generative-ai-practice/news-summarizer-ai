# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、Model Context Protocol (MCP) を、Linux Foundation傘下の新設されたAgentic AI Foundation (AAIF) に寄贈しました。
- AAIFは、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、Amazon Web Services (AWS)、Cloudflare、Bloombergが支援する基金です。
- MCPは、AIアプリケーションと外部システムを接続するためのユニバーサルなオープン標準として、過去1年でChatGPT、Geminiなどの主要AI製品や10,000以上のパブリックサーバーに広く採用されています。
- MCPはコミュニティ主導のRegistry、公式SDK、および非同期操作などの新機能を導入しており、Anthropicはオープンソース、コミュニティ主導、ベンダー中立性を保証することにコミットしています。
- この寄贈は、エージェントAIエコシステム全体で中立的かつオープンな標準を維持し、イノベーションを促進するというAnthropicのコミットメントを示すものです。
