# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、AIアプリケーションと外部システムを接続するオープン標準であるModel Context Protocol (MCP) を、Linux Foundation傘下のAgentic AI Foundation (AAIF) に寄付しました。
- Agentic AI Foundation (AAIF) は、Anthropic、Block、OpenAIが共同設立し、Google、Microsoft、Amazon Web Services (AWS) などが支援する組織です。
- MCPは、発表から1年で10,000以上の公開サーバー、ChatGPT、Gemini、Microsoft Copilotなどの人気AI製品、主要なクラウドプロバイダーに採用されるなど、広範な普及を達成しました。
- MCPは継続的に成長しており、Claudeの75以上のコネクタ、APIのTool SearchとProgrammatic Tool Calling機能、コミュニティ主導のRegistryと主要言語向けSDKなどが提供されています。
- この寄付は、MCPが中立的でオープンな標準であり続け、エージェントAIエコシステム全体のイノベーションを促進するというAnthropicのコミットメントを示すものです。
