# Donating the Model Context Protocol and establishing the Agentic AI Foundation

## Key Points
- Anthropicは、Model Context Protocol (MCP) を、Linux Foundation傘下に新設されたAgentic AI Foundation (AAIF) へ寄贈しました。AAIFはAnthropic、Block、OpenAIが共同設立し、Google、Microsoft、AWSなどが支援しています。
- MCPは、AIアプリケーションと外部システムを接続するための普遍的なオープン標準として1年間で広く採用され、10,000を超える公開サーバーで利用され、ChatGPT、Gemini、Microsoft Copilotなどの人気AI製品にも導入されています。
- AnthropicはMCPの成長に継続的に投資しており、Claudeの75以上のコネクタ提供や、生産規模のMCP展開を最適化するAPI機能（Tool Search、Programmatic Tool Calling）を最近発表しました。
- MCPがオープンソース、コミュニティ主導、ベンダー中立のままであることを保証するため、Linux Foundationへの寄贈が決定されました。
- Agentic AI Foundation (AAIF) は、戦略的投資、コミュニティ構築、オープン標準の共有開発を通じて、エージェントAIが透明性、協調性、公共の利益に基づいて発展することを目指しています。
