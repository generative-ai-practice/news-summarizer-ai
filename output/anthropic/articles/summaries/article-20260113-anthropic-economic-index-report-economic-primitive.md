# Anthropic Economic Index report: economic primitives

## Key Points
- Anthropicは、2025年11月におけるClaudeの利用状況を評価するため、ユーザーとAIのスキル、タスクの複雑さ、AIへの自律性の度合い、タスクの成功率、利用目的の5つの新たな経済プリミティブを導入しました。
- Claudeの利用は、Claude.aiでは拡張的利用が52%で優勢である一方、1P APIでは自動化が依然として支配的です。利用はコーディング関連タスクに集中し、上位10タスクでClaude.aiの会話の24%を占めます。
- Claudeはほとんどのタスクで成功するものの、人間が完了するのにかかる時間が長くなるようなより複雑なタスクでは成功率が低下する傾向があります。
- タスクの信頼性（成功率）を考慮すると、AIが米国労働生産性成長に与える年間寄与は1.8%ポイントから約1.0%ポイントに半減すると推定されます。AIは経済全体よりも高スキルなタスクに利用される傾向があり、一部の職種では仕事がデスクスキル化する可能性があります。
- Claudeの利用パターンには顕著な地理的差異があり、人間のプロンプトの教育レベルとAIの応答の教育レベルは0.92以上の高い相関を示しており、AIを効果的に活用するための人間側のスキルの重要性を強調しています。
