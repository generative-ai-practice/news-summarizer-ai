# Anthropic Economic Index: new building blocks for understanding AI use

## Key Points
- Anthropicは、AIの経済的影響を理解するための新しい「経済プリミティブ」（タスクの複雑性、スキルレベル、目的、AI自律性、成功率）を導入し、AIが仕事に与える変化のより詳細な分析を可能にした。
- Claudeは、より高度な教育を必要とする複雑なタスクにおいて最も大きなスピードアップ効果を発揮し、Claude.aiでは大学レベルのタスクで12倍の速度向上を示した。これはAIによる生産性向上が高スキルを要するタスクに集中していることを示唆している。
- Claudeのタスク成功率は、Claude.aiでは約19時間、APIでは約3.5時間かかるタスクで50%に達しており、ユーザーがタスクを細分化したり、成功しやすいタスクを選択することで実効的な作業時間軸が伸びる。
- AIの利用形態は国の経済発展段階によって異なり、GDPが高い国では仕事や個人利用が多い一方、低所得国では教育利用が中心である。AIは高学歴を要するタスクをカバーする傾向があり、一部の職業で「デスクスキル化」の可能性を示唆している。
- AIの普及による米国の労働生産性成長率は、タスクの信頼性を考慮すると年1.0〜1.2ポイント増加すると推定されており、これは1990年代後半から2000年代初頭の成長率に匹敵する重要な上昇である。
