# Anthropic Economic Index: new building blocks for understanding AI use

## Key Points
- Anthropicは、AI利用の経済的影響を理解するため、タスクの複雑性、スキルレベル、目的（仕事、教育、個人）、AIの自律性、成功という5つの「経済プリミティブ」を導入しました。
- Claudeは、より高い教育レベルを必要とする複雑なタスクにおいて、最も大きな作業高速化（Claude.aiで9倍、APIで12倍）をもたらしますが、タスク成功率を考慮すると生産性向上推定値は若干低下します。
- Claudeのタスク成功率は、人間が単独で行う場合のタスク時間との相関があり、Claude.aiでは約19時間、APIでは約3.5時間で50%の成功率を達成しますが、これはユーザーのタスク細分化などの要因による選択バイアスが含まれます。
- AIの利用はGDPの高い国では仕事や個人利用が多く、低い国では教育利用が中心です。AIがカバーするタスクは平均してより高い教育レベルを要するものであり、一時的に職業の「デスクキリング」効果をもたらす可能性を示唆しています。
- AIの普及による米国の労働生産性成長への寄与は、以前の推定である年間1.8ポイントから、タスクの信頼性を考慮するとClaude.aiで1.2ポイント、APIで1.0ポイントに再評価されました。
