# Anthropic Economic Index: new building blocks for understanding AI use

## Key Points
-   Anthropic Economic Indexは、AI利用を理解するための新たな「経済プリミティブ」（タスク複雑性、スキルレベル、目的、AI自律性、成功）を導入しました。
-   Claudeはより複雑なタスク（大学レベルの教育を要するタスク）において、より高い速度向上を示しますが、単純なタスクと比較して成功率はわずかに低下します。
-   AI利用はGDP per capitaが高い国では仕事や個人利用に、低い国では教育用途に集中しており、AIがカバーするタスクは平均してより高い教育レベルを要する傾向があります。
-   AIが米国の労働生産性成長率に与える影響は、タスクの加速のみを考慮すると年間1.8%の増加が見込まれますが、タスクの信頼性を考慮すると年間1.0〜1.2%に調整されます。
-   2025年を通じて、Claudeの利用はコンピュータ・数学関連タスクに集中し続けており、Claude.ai上でのAIとのインタラクションは自動化よりも人間の作業を「拡張」するパターンが主流となりました。
