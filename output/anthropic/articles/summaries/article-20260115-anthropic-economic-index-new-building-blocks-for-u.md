# Anthropic Economic Index: new building blocks for understanding AI use

## Key Points
- Anthropic Economic Indexの最新レポートでは、AI利用の経済的影響を理解するための新たな「経済的プリミティブ」が導入されました。これにはタスクの複雑さ、スキルレベル、目的、AIの自律性、成功の5つの指標が含まれます。
- Claudeはより複雑なタスク（大学卒レベル）において大幅な処理速度向上（12倍）を示し、AIによる生産性向上が高い人的資本を必要とする業務に集中している傾向が見られます。
- AIの利用目的は国の経済発展段階と相関し、高GDP国では仕事や個人利用が多い一方、低GDP国では教育目的での利用が主です。また、データ入力係や放射線科医など一部の職業がAIによって大きく影響を受ける可能性が示唆されています。
- タスクの信頼性を考慮した結果、AIの米国労働生産性成長への貢献度予測は年間1.0〜1.2%増に下方修正されましたが、これは依然として顕著な数値であり、1990年代後半から2000年代初頭の成長率に匹敵します。
- Claudeの利用は引き続き特定のタスクや地域に集中しているものの、Claude.aiではオートメーションよりも拡張（augmentation）が主要なインタラクションパターンとして優勢になっています。
