# Introducing Labs

## Key Points
- Anthropicは、Claudeの最先端機能に基づいた実験的な製品をインキュベートするチーム「Labs」を拡大しました。
- このアプローチにより、Claude Codeが6ヶ月で10億ドル規模の製品に成長し、Model Context Protocol (MCP)は月間1億ダウンロードを達成するなど、複数の成功した製品が生まれました。
- 元Instagram共同創設者であるMike KriegerがLabsに参加し、Ben Mannと共に製品開発を進めます。また、Ami Voraが製品組織を率い、Claude体験のスケールを担当します。
- Anthropicの社長Daniela Amodeiは、AIの急速な進歩に対応するため、Labsが実験的な製品の発見と、企業顧客や成長するユーザーベースへの責任あるスケーリングを可能にする重要な組織構造であると述べています。
- Anthropicは、AIの最先端で製品を構築し、新たなテクノロジーを慎重に形作る人材を積極的に募集しています。
