# Introducing Labs

## Key Points
- Anthropicは、Claudeの最先端能力を活用した実験的な製品を育成するため、「Labs」チームの拡大を発表しました。
- Instagramの共同創業者で、Anthropicの元CPOであるMike KriegerがLabsに加わり、Ami Voraは既存のClaude製品の拡大を担うプロダクト組織を率います。
- このアプローチによって、Claude Codeは6ヶ月で10億ドル規模の製品に成長し、Model Context Protocol (MCP) は月間1億ダウンロードを達成しました。
- Anthropicの社長であるDaniela Amodeiは、LabsがAIの進歩の速さに対応し、実験的な製品の発見と責任ある規模での展開をサポートする体制を整えるものだと述べています。
- Anthropicは、AIの最先端で製品を開発する人材を積極的に採用しています。
