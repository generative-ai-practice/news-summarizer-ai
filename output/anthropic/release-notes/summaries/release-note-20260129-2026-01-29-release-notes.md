## Updates (translated)
# 2026-01-29 リリースノート

- [構造化出力](https://platform.claude.com/docs/en/build-with-claude/structured-outputs)が、Claude APIのClaude Sonnet 4.5、Claude Opus 4.5、およびClaude Haiku 4.5で一般提供開始となりました。一般提供版には、拡張されたスキーマサポート、改善された文法コンパイルの遅延、およびベータヘッダー不要の簡素化された統合パスが含まれます。`output_format` パラメータは `output_config.format` に移動しました。既存のベータユーザーは、移行期間中もベータヘッダーを使用し続けることができます。構造化出力は、Amazon BedrockとMicrosoft Foundryでは引き続きパブリックベータ版として提供されます。
