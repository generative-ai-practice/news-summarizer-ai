# 2025年11月24日 リリースノート

- 2025年11月24日
- 私たちは、最大の能力と実用的なパフォーマンスを組み合わせた、最もインテリジェントなモデルである[Claude Opus 4.5](https://www.anthropic.com/news/claude-opus-4-5)をリリースしました。複雑な専門タスク、プロのソフトウェアエンジニアリング、高度なエージェントに最適です。以前のOpusモデルよりもアクセスしやすい価格帯で、ビジョン、コーディング、コンピュータ利用において飛躍的な改善が特徴です。詳細については、当社の[モデルと料金に関するドキュメント](https://platform.claude.com/docs/en/about-claude/models)をご覧ください。
- 私たちは、パブリックベータ版として[プログラムによるツール呼び出し](https://platform.claude.com/docs/en/agents-and-tools/tool-use/programmatic-tool-calling)を開始しました。これにより、Claudeはコード実行内からツールを呼び出すことができ、マルチツールワークフローにおけるレイテンシとトークン使用量を削減します。
- 私たちは、パブリックベータ版として[ツール検索ツール](https://platform.claude.com/docs/en/agents-and-tools/tool-use/tool-search-tool)を開始しました。これにより、Claudeは大規模なツールカタログからオンデマンドでツールを動的に発見し、ロードすることができます。
- 私たちは、Claude Opus 4.5向けにパブリックベータ版として[effortパラメータ](https://platform.claude.com/docs/en/build-with-claude/effort)を開始しました。これにより、応答の徹底性と効率性の間でトレードオフを行うことで、トークン使用量を制御できます。
- 私たちは、PythonおよびTypeScript SDKに[クライアントサイド圧縮](https://platform.claude.com/docs/en/build-with-claude/context-editing#client-side-compaction-sdk)を追加しました。これにより、会話コンテキストを要約によって自動的に管理します。使用時に。
