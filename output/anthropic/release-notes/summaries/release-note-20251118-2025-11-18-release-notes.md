# 2025年11月18日 リリースノート

- 2025年11月18日
- Microsoft FoundryでClaudeの提供を開始しました。これにより、Azure課金とOAuth認証を使用して、Azureのお客様にClaudeモデルをお届けします。拡張推論、プロンプトキャッシュ（5分および1時間）、PDFサポート、Files API、エージェントスキル、ツール利用を含むすべてのMessages APIにアクセスできます。詳細については、[Microsoft Foundryドキュメント](https://platform.claude.com/docs/en/build-with-claude/claude-in-microsoft-foundry)をご覧ください。
