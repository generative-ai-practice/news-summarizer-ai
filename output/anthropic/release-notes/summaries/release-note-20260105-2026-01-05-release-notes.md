# 2026-01-05 リリースノート

- Claude Opus 3モデル（`claude-3-opus-20240229`）を廃止しました。このモデルへのすべてのリクエストはエラーを返すようになります。大幅にインテリジェンスが向上し、コストが3分の1になった[Claude Opus 4.5](https://platform.claude.com/docs/en/about-claude/models/overview#latest-models-comparison)へのアップグレードをお勧めします。研究者の方は、[外部研究者アクセスプログラム](https://support.claude.com/en/articles/9125743-what-is-the-external-researcher-access-program)を通じて、API経由でClaude Opus 3への継続的なアクセスをリクエストできます。
