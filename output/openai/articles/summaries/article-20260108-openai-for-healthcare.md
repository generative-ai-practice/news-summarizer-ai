# OpenAI for Healthcare

## Key Points
- OpenAI for Healthcareは、高品質なケアの拡大、チームの管理業務削減、カスタム臨床ソリューションの強化を目的とした、HIPAAコンプライアンス要件をサポートする安全なAI製品です。
- ChatGPT for Healthcareは2026年1月8日から提供開始され、AdventHealth、Boston Children’s Hospitalなど主要医療機関に展開されており、GPT-5モデルを基盤に医療ワークフロー向けに設計されています。
- OpenAI APIは、数千の組織がHIPAA準拠の用途で活用しており、患者チャートの要約、ケアチームの調整、退院ワークフローなどの医療アプリケーション構築を可能にします。
- 全てのOpenAI for Healthcare製品はGPT-5.2モデルによって動作し、臨床医主導のテストと実世界評価を通じて最適化されており、HealthBenchやGDPvalなどのベンチマークで高い性能を示しています。
- 患者データおよびPHIは組織の管理下に置かれ、データレジデンシー、監査ログ、顧客管理の暗号化キー、およびOpenAIとの事業提携契約（BAA）によりHIPAA準拠の使用をサポートし、共有されたコンテンツはモデルのトレーニングには使用されません。
