# Datadog uses Codex for system-level code review

## Key Points
- Datadogは、複雑な分散システムの変更が全体に与える影響を評価する従来のコードレビューツールの課題に対し、OpenAIのコーディングエージェントCodexを導入しました。
- Codexは、コードベース全体と依存関係を推論することでシステムレベルのコンテキストをコードレビューに取り入れ、人間には見えにくいリスクを特定します。
- 過去のインシデントを再検証した結果、Codexは調査したインシデントの約22%において、人間が見落としたであろうフィードバックを提供し、インシデント防止に役立つことが確認されました。
- エンジニアはCodexのコメントを「ボットノイズ」ではなく、システム間の相互作用やAPI契約の変更に伴うリスクなど、高信号なフィードバックとして評価しています。
- 現在、1,000人以上のエンジニアがCodexを日常的に利用しており、コードレビューの焦点がエラー検出から、信頼性強化と大規模なシステムにおけるリスク特定へとシフトしています。
